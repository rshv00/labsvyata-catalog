# Azure Function Host (Static)

This folder contains a minimal Azure Functions host that serves the static export generated by the root Next.js app.

## Deploy flow

1. Build static site in the root project:

```bash
npm run build
```

2. Copy static files into this host:

```bash
rsync -a --delete out/ azure-function-host/out/
```

3. Install host dependencies:

```bash
cd azure-function-host
npm install
```

4. Zip and deploy to Azure Function App:

```bash
zip -qr /tmp/labsvyata-function-deploy.zip .
az functionapp deployment source config-zip --resource-group <rg> --name <app> --src /tmp/labsvyata-function-deploy.zip
```

The host uses `routePrefix: ""`, so routes are served from the domain root.
